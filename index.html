1. 项目名称

2020年中国各省常住人口与城镇化水平空间可视化系统

2. 数据来源

  • GeoScene Online WebMap：5d6e5d9225294277b6317dea60beab2e
  • 字段：SUM_2020 常住人口；2020_城 城镇化率

3. 实验要求对照

  • 三大区域：页眉 / 地图区 / 底图库与图层区（左侧面板）
  • 地图显示：底图 + GeoScene Online 专题内容
  • 控件：BasemapGallery、LayerList、Legend、Search、ScaleBar

4. 实用功能

  • 数据概览（总人口/平均城镇化）、Top10 排行、快速定位省份
  • 量测工具（距离/面积）
  • 地图截图/导出 PNG
  • 重置：一键恢复默认状态（回到默认视图、清除量测、收起侧栏并回到默认面板）

5. 部署与提交

部署到 GitHub/Gitee Pages，提交 URL + 截图（含页眉、地图、侧栏控件）。
